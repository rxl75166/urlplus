# urlplus

This is a simple bash script for getting passive urls from a gau, gauplus, waybackurls from a Multiple Subdomain.

How to use
<div>git clone https://github.com/rxl75166/urlplus.git<div>
<div>cd urlplus<div>
chmod +x urlplus.sh
./urlplus.sh subdomains.txt
